Проект на тему "cжатие изображений SVD и JPEG"

Это учебный проект на языке C. В нем реализовано сжатие BMP-изображений двумя способами: через SVD и через JPEG-подобный алгоритм. Программа также считает метрики PSNR и SSIM, чтобы сравнить качество результата.
Основная цель проекта — показать, как работают разные методы сжатия, и сравнить их по качеству изображения, размеру файла и времени работы.

Что умеет программа:
Программа может сжимать изображение методом SVD, восстанавливать BMP из файла .svd, сжимать изображение JPEG-подобным способом, считать PSNR и SSIM, а также запускать сравнение для нескольких параметров сжатия.

SVD-сжатие:
В SVD-сжатии изображение представляется как матрица пикселей. Эта матрица раскладывается на несколько матриц, после чего сохраняется только часть самых важных значений. За качество отвечает параметр k. Чем меньше k, тем сильнее сжатие и хуже качество. Чем больше k, тем лучше качество, но больше размер файла.

Пример запуска SVD:
image_compressor svd input.bmp output.svd 20

Чтобы получить BMP обратно:
svd_to_bmp output.svd output.bmp

JPEG-сжатие:
JPEG-подобный алгоритм делит изображение на блоки 8 на 8 пикселей, применяет DCT, выполняет квантование и восстанавливает изображение обратно. За качество отвечает параметр quality. Чем выше quality, тем лучше качество и больше размер файла.

Пример запуска JPEG:
image_compressor jpeg input.bmp output.jpg 60

Для сравнения можно использовать режим compare:
image_compressor compare input.bmp results 5,10,20,40 20,40,60,80

После запуска программа сохраняет результаты в папку results и создает файл summary.csv. В нем записываются значения PSNR, SSIM, степени сжатия и времени работы.

Сборка:
Для сборки нужен GCC.

На Windows можно использовать файл build.bat:
build.bat

На Linux или macOS можно собрать проект через make:
make

Для очистки файлов сборки:
make clean

Структура проекта:
include/ — заголовочные файлы
src/ — основной код программы
scripts/ — дополнительные скрипты
Makefile — сборка для Linux/macOS
build.bat — сборка для Windows
app.bmp — пример изображения
README.md — описание проекта

Основные файлы кода:
src/main.c — запуск программы и обработка команд
src/bmp.c — работа с BMP-файлами
src/svd.c — SVD-сжатие
src/jpeg_like.c — JPEG-подобное сжатие
src/jpeg_writer.c — запись JPEG-файла
src/metrics.c — расчет PSNR и SSIM
src/compare.c — сравнение методов

Метрики:
PSNR показывает, насколько сильно сжатое изображение отличается от исходного. Чем выше PSNR, тем лучше.
SSIM показывает визуальное сходство изображений. Значение ближе к 1 означает лучшее качество.
Ratio показывает степень сжатия.
Time показывает время работы алгоритма.

Вывод:
В проекте реализованы два метода сжатия изображений: SVD и JPEG-подобный алгоритм. Программа позволяет менять параметры сжатия и сравнивать результат по качеству, размеру файла и времени работы.
